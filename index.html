Canal-Licitação
HTML - CSS - JS
PHP - BOOTSTRAP

O projeto foi inspirado no sistema Canal de licitação, onde foi criado com o intuito de gerenciar licitações.
esse sistema de licitação foi realizado com o framework ScriptCase(Ferramenta Lowcode).
Ferramenta que limitava algumas funções em expecifca utilizando o metodo POO.

Nesse projeto foi utilizado HTML, CSS, JS e PHP com a ajuda do framework BootsTrap V5.0.
Utilizando uma criptografia de senha para acesso do login, e o metodo POO em php para realizar conexões com DataBase e orientação do sistema.
sistema base: https://canaldelicitacoes.com.br/
